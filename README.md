# hillel-09012025
Hillel educational repository.

Welcome!

1. Be sure that you are on "main"

```git switch main```

2. Update your "main"

```
git fetch
git pull
```

3. Create a new branch for the task

```git checkout -b BRANCH_N```

4. Do the task and add the files

``` git add . ```

5. Create commit

``` git commit -m "Some message" ```

6. Push

``` git push origin BRANCH_N ```
